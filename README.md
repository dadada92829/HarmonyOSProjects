# HarmonyOSProjects

个人 HarmonyOS 应用合集。

| 项目 | 描述 |
|------|------|
| `project1_24points` | 24点数学游戏 |
| `project2_gomoku` | 五子棋，支持AI对弈（简单/中等/困难） |
| `project3_accountbook` | 记账本，支持收支管理、AI理财建议 |
| `project_final_notebook` | 多功能笔记App，支持AI摘要/思维导图/OCR/语音 |

## 使用方式

每个项目用 [DevEco Studio](https://developer.huawei.com/consumer/cn/deveco-studio/) 打开对应目录即可运行。

> 含 AI 功能的项目需要自行申请 API Key，将 `.ets.template` 复制为 `.ets` 并填入密钥。
